# 数据字典（M0骨架）

## 1. 角色枚举
- admin（管理员）
- coach（班主任/教练）
- student（学生）

## 2. 学历枚举（示例）
- 专科
- 本科
- 硕士
- 博士

## 3. 政治面貌枚举（示例）
- 群众
- 共青团员
- 中共党员
- 中共预备党员

## 4. 应届/社招
- campus（应届）
- social（社招）

## 5. 地区偏好
- 省/市两级，按优先级数组保存：`["北京","上海","广东-深圳"]`
- 规范化规则：省份与城市之间用 `-` 连接，直辖市只写城市名。

## 6. 专业标准化（占位）
- 字段：`major`（原始输入）、`major_standard`（标准化名称）
- 规则：先映射到教育部学科门类/专业大类（MVP 前补全映射表）

> 说明：详细枚举与专业映射在 MVP 前补齐。
